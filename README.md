# A spatially defined glial-tip cell niche around tumor boundary promotes mesenchymal tumor states and immune exclusion in microsatellite-stability colorectal cancer


## Introduction

We developed a cross-platform classifier TiGELR, a machine learning model via Tip cells-Glial cells-EMT II-related tumor cells Ligand-Receptor, accurately distinguishes MSS from MSI and stratifies immunotherapy outcomes

We applied TiGELR to classify MSI/MSS status using untreated CosMx, scRNA-seq, and bulk RNA-seq datasets. TiGELR effectively distinguished MSI and MSS phenotypes across all platforms, including single-cell spatial transcriptomics (AUC = 0.82), single-cell RNA-seq (AUC = 0.91), and bulk RNA-seq (AUC = 0.77). Given that MSS CRC patients are generally more likely to be non-responders to immune checkpoint blockade (ICB) therapy, we explored whether TiGELR could predict ICB treatment response. Using a dataset of 40 ICB-treated samples with scRNA-seq data, TiGELR significantly differentiated between responders and non-responders (AUC = 0.82)
